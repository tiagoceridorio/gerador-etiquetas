# Gerador de Etiquetas

Aplicação web para criar e imprimir etiquetas artesanais para produtos como sais de banho, jellies e outros cosméticos naturais.

**[Acessar o Gerador](https://tiagoceridorio.github.io/gerador-cartoes/)**

## O que faz

Gera cartões de etiqueta no formato 50×80mm com design ornamental vintage, prontos para impressão em folha A4. Cada cartão possui frente (nome, ingredientes, ilustração e datas) e verso (modo de uso), com o verso espelhado por linha para facilitar a montagem após o corte.

## Funcionalidades

- **Editor visual com pré-visualização ao vivo** — veja frente e verso do cartão enquanto edita
- **Modelos rápidos** — presets para produtos comuns (Sal Estimulante, Sal Calmante, Jelly Calmante, etc.)
- **Campos personalizáveis** — tipo, nome, subtítulo/ingredientes, datas de fabricação e validade, modo de uso
- **Ilustração por imagem ou emoji** — upload de imagem botânica ou uso de emojis
- **Banner escuro** — opção visual para linhas noturnas ou edições premium, com cor configurável
- **Lote de produtos** — cadastre vários produtos com quantidades diferentes antes de imprimir
- **Impressão em A4** — 12 cartões por página, com páginas de frente e verso intercaladas
- **Design responsivo** — funciona em desktop e dispositivos móveis

## Como usar

1. Acesse a aplicação no navegador
2. Escolha um modelo rápido ou preencha os campos manualmente
3. Clique em **Salvar produto** para adicionar ao lote
4. Repita para outros produtos, ajustando a quantidade de cada um
5. Clique em **Imprimir** para gerar as páginas de etiquetas

## Tecnologias

Aplicação 100% client-side — um único arquivo HTML com CSS e JavaScript embutidos, sem dependências externas ou backend. Usa Google Fonts (Playfair Display, Lora, Dancing Script, Great Vibes, Berkshire Swash, Cormorant Garamond) e a fonte customizada Breathing.
